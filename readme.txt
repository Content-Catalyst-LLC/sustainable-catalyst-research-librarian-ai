A site-scoped Research Librarian for Sustainable Catalyst with source-aware routing, route notes, provider options, and a knowledge indexer/admin crawl dashboard.

== Description ==

The Research Librarian helps visitors choose the right Sustainable Catalyst route: Knowledge Library, Platform, Demos, Decision Studio, Workbench, Catalyst modules, Methodology, Feature Suggestions, or GitHub repositories.

v3.2.0 adds a Knowledge Indexer and Admin Crawl Dashboard for route coverage, source records, metadata warnings, stale records, and index export.
