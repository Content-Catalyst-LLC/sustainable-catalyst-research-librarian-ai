Site-scoped routing layer for Sustainable Catalyst with deterministic fallback, optional Gemini/OpenAI, route notes, and module-aware navigation.

== Description ==

The Research Librarian helps visitors choose the right Sustainable Catalyst route: Knowledge Library, Workbench, Decision Studio, Platform Demos, modules, methodology, GitHub, support, consulting, or feature suggestions.
