A site-scoped Research Librarian for Sustainable Catalyst with source-aware routing, route notes, source cards, handoff payloads, saved route sessions, admin analytics, a knowledge indexer, optional Gemini embeddings for hybrid retrieval, governance, security, observability, curation, contracts, and polished public answer UX.

== Description ==

The Research Librarian helps visitors choose the right Sustainable Catalyst route: Knowledge Library, Platform, Demos, Decision Studio, Workbench, Catalyst modules, Methodology, Feature Suggestions, or GitHub repositories.

v4.6.0 adds public answer UX, source cards, confidence badges, reason-code chips, low-confidence states, and a route action center for visitor-facing answers.
