# v5.3.1 — Activation Fatal Guard and Stable Plugin Folder Fix

This patch fixes the most likely v5.3.0 activation fatal: two active plugin folders declaring the same Research Librarian PHP classes.

The v5.3.1 plugin zip is packaged with the stable WordPress folder slug:

```text
sustainable-catalyst-research-librarian-ai
```

Recommended recovery steps:

1. Deactivate older versioned Research Librarian plugin copies.
2. Upload `sustainable-catalyst-research-librarian-ai-plugin-v5.3.1.zip`.
3. Activate v5.3.1.
4. Delete older inactive versioned copies after confirming v5.3.1 works.

All v5.3.0 article map integration features are preserved.
