# Research Librarian v4.6.0 — Public Answer UX, Source Cards, and Route Action Center

v4.6.0 upgrades the public answer experience for the Sustainable Catalyst Research Librarian.

## Goal

Make the public response feel like a finished institutional product rather than a plain chatbot answer.

## Features

- Recommended route card
- Matched source cards
- Confidence badge
- Reason-code chips
- Ambiguity and low-confidence states
- Route Action Center
- Workbench and Decision Studio handoff actions
- Route note copy/download
- Handoff JSON download
- Session save and feedback actions
- Feature Suggestions fallback

## Public answer structure

1. Recommended route
2. Why this route fits
3. Source-aware support
4. Confidence and ambiguity
5. Next actions
6. Boundary note

## Boundary

The answer UX is public-safe and does not expose admin diagnostics, keys, raw logs, or privileged exports.
